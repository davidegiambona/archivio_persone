# App per archiviazione indagati e pregiudicati stile poliziesco
- archiviazione possibile sia in locale che in un server db
- verifica sui dati in fase di compilazione e modifica
- possibilità di modificare dati delle persone inserite
- possibilità di eliminare eventuali persone
- possibilità di allegare per ogni utente un pdf che ne descriva informazioni aggiuntive, precedenti, cause, fedine penali etc...
- possibilità di ricerca eventuali persone per nome, cognome e CUI.
- possibilità di eseguire backup di storage per mantenere una sicurezza dei dati nel caso di modifiche o cambiamenti al database
